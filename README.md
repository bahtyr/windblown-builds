# Windblown Builds (WIP)

A build designer site for **[Windblown](https://windblown.game/)**, inspired by tools like Mobalytics.gg for LoL / TFT.

The goal is to let users explore gifts/weapons/abilities, create builds and share them with others.

###### Purpose

This is a learning project to gain experience on modern frontend architecture (React/TS) and AI-assited development.

Previously, I built [ChampsDB](https://github.com/bahtyr/champsdb) for League of Legends using vanilla JS. </br>
This time I want a proper framework-based approach.

###### Data
Special thanks to the [Windblown Wiki](https://windblown.wiki.gg/) for providing detailed and structured data that makes this possible.
